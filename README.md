# Chatbot using RAG and LLAMA Index

## Overview
This repository contains Python code implementing a chatbot using RAG (Retrieval-Augmented Generation) model along with LLAMA Index (Linguistic Link Adaptation Model for Information Retrieval) for improved response generation in conversational AI systems. RAG integrates a retriever component with a generative model, allowing the chatbot to generate responses based on retrieved passages from a knowledge source.

## Features
- Utilizes RAG architecture for response generation, combining retriever and generative model.
- Enhances response quality through LLAMA Index, a metric that helps in selecting informative passages for response generation.
- Easily adaptable codebase for implementing custom conversational AI systems or chatbots.
- Well-documented Python code for understanding and extending the functionality.

## Dependencies
- Python 3.x
- Transformers library by Hugging Face: for RAG model implementation.
- Hugging Face Datasets library: for accessing datasets for training and evaluation.
- PyTorch: Deep learning library used as a backend for RAG model.
- Other standard Python libraries: NumPy, pandas, etc.
